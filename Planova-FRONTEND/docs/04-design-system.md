# Design System

## Colores

### Base
- Fondo: oscuro espacial
- Glass: blanco translúcido

### Estados
- Verde → tareas completadas
- Azul → tareas en proceso
- Amarillo → tareas pendientes
- Rojo → tareas vencidas

## Bordes
- Muy redondeados

## Sombras
- Glow exterior
- Glow interno (inset)

## Blur
- Alto (clave del glass)

## Tipografía
- Moderna, amigable
- Estilo handwritten para títulos (como en diseño)
