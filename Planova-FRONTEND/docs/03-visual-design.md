# Visual Design

## Estilo general
- Glass Liquid
- Bordes suaves y brillantes
- Transparencias con blur
- Glow dinámico

## Fondo
- Espacio / galaxia con estrellas
- Gradientes oscuros

## Elementos clave

### Cards
- Efecto vidrio
- Bordes iluminados
- Sombra difusa

### Inputs
- Transparente
- Glow en focus

### Botones
- Redondeados
- Efecto "flotante"

### Mascota (muñequito)
- Presente en login
- Elemento amigable
- Refuerza identidad visual

## UI Constraint (Obligatorio)

La implementación debe replicar EXACTAMENTE el diseño visual de las imágenes provistas en `/docs/assets`.

No se permiten variaciones en:

- Layout
- Colores
- Estilo glass (glassmorphism + liquid)
- Distribución de elementos
- Espaciados
- Tipografía visual aproximada

## Justificación

El diseño visual es parte central del producto y constituye su principal diferenciador.

Cualquier desviación afecta la identidad de Planova.
