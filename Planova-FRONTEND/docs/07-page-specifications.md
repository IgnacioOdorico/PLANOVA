# Project Page (Kanban)

## Elementos
- Nombre del proyecto
- Descripción
- Columnas (con titulo de la columna y un "agregar tarea" abajo para sumar una tarea nueva a la columna)
- Tareas de colores
- Botón agregar columna
- las tareas tienen: titulo de tarea. descripcion de la tarea. estado, prioridad y fecha de vencimiento cliquiable para cambiarlo. boton "notas" para ver las notas

## Layout
- Horizontal scroll
- Columnas separadas
