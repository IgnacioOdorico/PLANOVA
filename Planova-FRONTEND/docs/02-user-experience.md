# User Experience

## Principios
- Todo debe sentirse fluido
- Nada debe ser brusco
- Cada acción tiene feedback visual

## Sensaciones buscadas
- Calma
- Control
- Claridad
- Estética moderna

## Flujo principal

1. Usuario entra → pantalla login
2. Crea cuenta / inicia sesión
3. Ve lista de proyectos
4. Entra a proyecto
5. Gestiona tareas tipo Kanban
