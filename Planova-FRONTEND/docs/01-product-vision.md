# Product Vision

Planova es una aplicación de gestión de tareas con una estética diferencial basada en:

- Glassmorphism
- Liquid UI
- Ambientación espacial (galaxia)

## Objetivo
Crear una herramienta que el usuario QUIERA usar todos los días.

## Diferencial
- UI emocional (no solo funcional)
- Sensación de "producto premium"
- Feedback visual constante
