# Assets

## Ubicación
/docs/assets/

## Descripción
Esta carpeta contiene TODOS los recursos visuales base utilizados como referencia obligatoria para el desarrollo del frontend.

## Archivos principales
- home.png → vista de proyectos
- login.png → pantalla de registro/login
- proyecto.png → vista kanban

## Reglas de uso
- Estas imágenes son la fuente de verdad del diseño
- El frontend debe replicar fielmente cada detalle visual
- No se deben reinterpretar estilos

## Notas
- Incluye en una carpta "/docs/assets/img para utilizar/" la mascota (muñequito) con variasiones de colores, titulo del home con mascota, y un logo para implementar en algun lugar
