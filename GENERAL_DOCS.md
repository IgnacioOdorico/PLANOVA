# Planova — Documentación General

## 🚀 ¿Qué es Planova?
Planova es una herramienta de gestión de tareas basada en la metodología **Kanban**, diseñada para usuarios que buscan organización con una estética moderna (Glassmorphism) y una experiencia fluida. 

La aplicación permite a los usuarios gestionar proyectos enteros, segmentados en columnas personalizables y tareas con prioridades y fechas de vencimiento.

---

## 🛠 Flujo de Usuario Principal

### 1. Acceso y Seguridad
- El usuario debe registrarse para obtener una cuenta personal.
- Todo el contenido de Planova es **privado** por defecto. Gracias a la arquitectura de "Ownership", ningún usuario puede ver o editar proyectos de otros, asegurando total confidencialidad.

### 2. Gestión de Proyectos (Dashboard)
- Al ingresar, el usuario accede a su panel de control con todos sus proyectos activos.
- Se muestran estadísticas rápidas, como la cantidad de tareas por proyecto.

### 3. El Tablero Kanban
Es el corazón de Planova. El usuario puede:
- **Crear Columnas:** Para definir etapas (ej: "Pendiente", "En curso", "Bloqueado").
- **Gestionar Tareas:** Crear tareas con títulos, descripciones, etiquetas de prioridad y fechas límite.
- **Drag & Drop:** Mover tareas entre columnas de forma intuitiva arrastrándolas con el mouse.
- **Estado Inteligente:** Si una tarea tiene fecha de vencimiento y esta pasa, el sistema la marcará automáticamente como "Vencida" resaltándola visualmente.

### 4. Perfil y Notas
- Cada usuario puede gestionar sus datos personales.
- Las tareas permiten agregar notas detalladas o comentarios para llevar un registro del progreso.

## 💡 Sugerencias de Mejora a Nivel Producto

1.  **Colaboración:** Actualmente es monousuario. Implementar un sistema de invitación por email para compartir proyectos con otros colaboradores sería el siguiente gran paso.
2.  **Notificaciones:** Alertas vía email o notificaciones push en el navegador cuando una tarea esté por vencer.
3.  **Temas Personalizados:** Permitir al usuario cambiar el color del gradiente de fondo o elegir temas (Oscuro/Claro/Espacial).
4.  **Exportación de Datos:** Botón para exportar el tablero a PDF o CSV para reportes de trabajo.
